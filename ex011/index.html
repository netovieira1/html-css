Imagens dinâmicas

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e dimunua o tamanho do seu navegador.

Usando a tag 'picture' pode-se adicionar uma imagem com a tag 'img' Depois usa-se a tag 'source:media:type',no parametro de 'media' usa-se o max-width(vem o min como padrão), e deixa '1050px' que é o tamanho máximo de uma imagem de 'tamanho m', mais que isso altomaticamente a imagem será 'tamanho g' (imagem original) na barra de rolagem do site, e se for 'max width 750', automaticamente muda para a 'imagem p'.

Reproduzindo áudio

Vamos aprender a reproduzir áudios em HTML5.

Utilizamos a tag 'audio' e em src o nome do arquivo, o parâmetro 'controls' mostra as opções de volume, pause etc. O parâmetro 'autoplay' toca o áudio assim que a página é carregada
