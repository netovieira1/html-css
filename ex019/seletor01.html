Aprendendo HTML

HTML básico

Quando apenas uma única tag precisa ser modificada (ex:h1) sem alterar as outras, utiliza-se o parâmetro 'id' dentro da tag que precisa ser alterada e damos uma identificação a ela (ex: id="principal"). E para que essa tag seja modificada em CSS, utiliza-se o símbolo '#'. Ex: h1#principal(o h1 antes da # não é obrigatório). Só pode existir um elemento por 'id', mas podem existir vários elementos com o 'class'

Para alterar mais de uma tag ao mesmo tempo utiliza-se o parâmetro 'class'(ex:class="basico"). Para essas tags serem modificadas em CSS, utiliza-se o ' . '(ponto). Ex: .basico

Em HTML é id = em CSS = #. Em HTML é class = em CSS é ' . ' (ponto).

Podem ser utilizadas mais de uma classe ou classe com id na mesma tag. Exemplo para mais de uma classe: h1 class="basico intermediario"(dentro das aspas separadas sem vírgulas).

Exemplo de classe com id: h1 class="basico" id="principal class="basico". A id sobrepõe a class.
