Usando links externos

Você pode acessar link um externo, como por exemplo, um repositório público de github.

Usando a tag 'a'(anchor), o parâmetro 'href' para referenciar o link e os parâmetros target ="_blank" e rel ="external" para abrir em uma nova aba sem que o seu site seja fechado. O parâmetro target ="self" indica que aquela página que irá ser clicada, pertence ao meu site.

Você pode acessar também um link do YouTube.
