Testando Gradientes

Para criar gradientes no background, utiliza-se a tag 'background-image: linear-gradient'(to right, white, blue). Da esquerda para a direita branco, até ficar azul

o '*{} = Configurações globais do CSS. Deixar a altura 100%(height) é necessário para usar gradientes de cima pra baixo ou baixo pra cima.
