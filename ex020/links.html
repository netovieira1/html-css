Pseudo-Classes são palavras que definem estados de elementos dentro da HTML. São representadas por ' : '(dois pontos)

Exemplo de Pseudo-Classe em CSS é a tag 'a' utilizada para links. Ex: a:visited{color: red;} altera a cor do link já visitado.

a:hover{color: black; text-decoration: underline;} altera a cor do link ao passar o cursor do mouse em cima e fica sublinhado a:active {color: blueviolet;} muda a cor ao passar segurar o click do mouse sobre o link.

Pseudo-Elementos podem agir nas classes, elementos, id etc. Eles mexem diretamente no conteúdo periférico do elemento e são representados por ' :: '(dois pontos duplo).

Exemplo de Pseudo-Elementos: a::after {content: ' link'} adiciona a palavra 'link' depois do link.
